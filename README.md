# FreePBX extensions and queues status realtime monitoring

Simple setup: Go to Admin > Module Admin > Upload Modules to install the module.

Or upload files to /var/www/html/admin/modules/fmonitor

chown asterisk:asterisk /var/www/html/admin/modules/fmonitor

fwconsole ma install fmonitor

fwconsole reload
